🔧 Troubleshooting
  • Problem: Page stuck on "Processing Your Payment"
    • ✓ Open browser console (F12)
    • ✓ Look for "Saving enrollment data to database..."
    • ✓ Check for fetch errors
  • Problem: Success page shows but database is empty
    • ✓ Check console for JavaScript errors
    • ✓ Verify process_enrollment.php is accessible
    • ✓ Check PHP error logs
  • Problem: Emails not in /emails/ folder
    • ✓ Check that folder exists and is writable
    • ✓ Check config.php MAIL_METHOD setting
    • ✓ Check PHP permissions
